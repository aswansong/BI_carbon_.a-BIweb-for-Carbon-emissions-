注：当前文件夹是主要文件夹！！！

data 是本平台使用数据的数据处理中心，内含有CEADS官方授权数据，jupyter使用文件等等。

management内是我配置的django命令的具体处理文件——python manage.py addfakeorders 2000.

migration文件夹中是数据库迁移文件

static文件夹是开源的百度amis框架和bootstrap以及本平台所使用的图片视频素材。

templates文件夹中是主要的页面html配置

行政规划数据是获取为国省市具体坐标数据的相关代码

data_analysis.py是图表数据处理中心（相关代码）

models.py 是使用djangoORM模式的相关数据库表设计、

tasks.py 是异步任务系统所执行的代码

urls.py 是平台的前后端传输接口所在文件

views.py是平台主要的后端处理中心代码文件
